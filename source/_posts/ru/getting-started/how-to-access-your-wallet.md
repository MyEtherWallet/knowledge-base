# __Как получить доступ к кошельку__
##### <i>Эта статья предназначена для тех, у кого уже есть ETH-кошелек. Если вы хотите создать новый кошелек, ознакомьтесь, пожалуйста, с этой [статьей].</i>

## MyEtherWallet (MEW) предлагает множество способов доступа к Ethereum-кошельку с помощью опции "Доступ к моему кошельку" на главной странице.

## __Способы доступа, от наиболее к наименее рекомендованным:__

<br>

* Аппаратные кошельки ([Ledger Nano S](https://www.ledger.com/?r=fa4b), [Trezor](https://trezor.io/?offer_id=12&aff_id=2029), [Digital Bitbox](https://shiftcrypto.ch/?ref=mew), [Finney](http://shop.sirinlabs.com/?rfsn=2397639.54fdf&utm_source=refersion&utm_medium=affiliate&utm_campaign=2397639.54fdf) и т. д.)
* MEW wallet, официальное мобильное приложение MEW для iOS и Android, и WalletLink / WalletConnect
* Расширения для браузера с поддержкой Web3 (MEW CX, MetaMask и т.д.)
* Программный доступ
    * Файл-хранилище ключа / JSON-файл + пароль
    * Мнемоническая фраза + дополнительное слово (необязательно)
    * Закрытый ключ

## __Наиболее рекомендуемые способы:__

## __Аппаратные кошельки__

Аппаратные кошельки используют самые сильные меры безопасности для шифрования закрытого ключа на самом устройстве, защищая вас от фишинга и хакеров. Пользователи большинства аппаратных кошельков получают не сам закрытый ключ, а мнемоническую фразу из 24 слов, нужную для восстановления. Также на устройствах обычно есть пин-код или пароль для разблокировки устройства, а вся информация о каждом кошельке хранится на самом физическом устройстве.

Чтобы узнать больше о доступе к MEW с помощью аппаратных кошельков, смотрите нашу категорию [Миграция на/с MEW](/@@@@@@/hardware-wallets/).

## __MEWwallet__
Это наше полнофункциональное мобильное приложение MEW для смартфонов iOS и Android. Оно простое в использовании, быстрое и безопасное. При создании кошелька с помощью MEW wallet ваш закрытый ключ хранится в зашифрованном виде в безопасном локальном хранилище на вашем телефоне. Для резервного копирования вам дается мнемоническая фраза (как на аппаратных кошельках выше). Кошелек также имеет PIN-код или пароль, который используется для выполнения входа на вашем устройстве.

Чтобы узнать больше о доступе к MEW с помощью MEW wallet, ознакомьтесь с нашим [Руководством пользователя MEW wallet](/@@@@@@/mewwallet/mewwallet-user-guide/).

##### __

## __WalletLink / WalletConnect__
WalletLink это протокол соединения, используемый мобильным приложением Coinbase Wallet. WalletConnect это другой протокол подключения, который уже используется более чем 20 мобильными кошельками, включая Trust Wallet, MetaMask, Coinomi и др. Интеграция поддержки соединения этих кошельков с MEW дает пользователям возможность изучать весь Ethereum с множества разнообразных точек доступа.

Эти кошельки имеют свои собственные процессы подключения, но он всегда сводится к тому же действию - сканирование QR-кода с помощью вашего приложения для подключения к MEW.

## __Расширение для браузера (Web3):__
#### __Что такое [Web3](https://www.mewtopia.com/the-essential-wallet-guide-part-4/)?__



## __MEW CX__
MEW CX — это официальный кошелек-расширение для Chrome от MyEtherWallet с поддержкой Web3. Расширение предлагает полную функциональность веб-версии MEW с удобством расширения на панели инструментов. Кроме того, оно предоставляет возможность взаимодействия с сетевыми Dapp-приложениями, как, например, CryptoKitties.

MEW CX вступает в конфликт с любыми другими кошельками Web3, такими как MetaMask или Brave Browser. Убедитесь, что все остальные Web3 кошельки отключены перед использованием MEW CX.


## __Чтобы подключить MEW к MEW CX__

**Шаг 1.** Установите кошелек MEW CX с помощью нашего расширения для Chrome.

**Шаг 2.** Откройте расширение и выберите "Добавить кошелек", чтобы создать новый кошелек или добавить существующий.

**Шаг 2.** После сохранения кошелька направляйтесь на сайт MEW. На главной странице выберите "Доступ к кошельку".

**Шаг 3.** Выберите вариант "Расширение для браузера" для подключения.

**Шаг 4.** Прочитайте и примите "Условия использования", а затем выберите "Доступ к кошельку".

**Шаг 5.** Подтвердите кошелек, который вы хотите подключить, во всплывающем окне MEW CX или выберите "Временный аккаунт" для временного кошелька.

**Шаг 6.** Готово!




## __MetaMask__
Это расширение для Chrome, позволяющее создать кошелек на базе ethereum, который хранит ваш закрытый ключ в зашифрованном виде в самом расширении. Оно также генерирует мнемоническую фразу из 12 слов для восстановления. MetaMask позволяет пользователям давать разным кошелькам имена, просматривать баланс средств в расширении, и дает возможность при желании импортировать ранее созданный кошелек путем закрытого ключа.



## __Для подключения MEW с помощью MetaMask:__
#### __Предполагается, что у вас уже есть кошелек MetaMask.__

**Шаг 1.** Войдите в ваш кошелек MetaMask с помощью их расширения для Chrome.

**Шаг 2.** На главной странице MEW выберите "Доступ к кошельку".

**Шаг 3.** Выберите вариант "Расширение для браузера" для подключения.

**Шаг 4.** Прочитайте и примите "Условия использования", а затем выберите "Доступ к кошельку".

**Шаг 5.** Подтвердите подключение во всплывающем окне MetaMask.

**Шаг 6.** Готово!

## __Эти методы не рекомендуются:__

## __Хранилище ключей / JSON + пароль__
Файл Keystore/JSON — это файл, который содержит зашифрованную версию закрытого ключа. Этот ключ зашифрован с помощью выбранного пользователем пароля. Фактически изменить пароль этих файлов невозможно. Единственный способ изменить пароль — это создать абсолютно новый файл Keystore, что можно сделать с помощью закрытого ключа в нашем расширении MEW для Chrome (MEW CX). Файлы Keystore не должны открываться. Их нужно использовать только в опции "Хранилище ключей / JSON" на нашем сайте и тогда, когда запрашивают ваш файл JSON. Прочитать подробнее о функциональности файла хранилища ключей / JSON-файла можно [здесь](/@@@@@@/security-and-privacy/what-is-a-keystore-file/).



## __Подключение с помощью файла хранилища ключей / JSON и пароля:__
#### __Это не рекомендуемый способ подключения. Мы настоятельно рекомендуем [использовать MEW оффлайн](/@@@@@@/offline/offline-mew-looks-weird/) для этого варианта.__

**Шаг 1.** На главной странице MEW выберите "Перейти к моему кошельку".

**Шаг 2.** Выберите вариант "Программный доступ" для подключения, затем выберите "Файл-хранилище ключа" и нажмите "Продолжить".

**Шаг 3.** Найдите ваш файл хранилище/JSON во всплывающем окне. Он должен начинаться с "UTC--".

**Шаг 4.** Введите пароль и нажмите "Доступ к кошельку".

**Шаг 5.** Готово!



## __Мнемоническая фраза (+ доп. слово)__
Это список из 12 или 24 слов, которые генерируются и предоставляются пользователям при выборе определенных вариантов создания кошелька (MEWconnect, MetaMask, аппаратные кошельки, кошельки JAXX и т. д.). Иногда у этих фраз есть дополнительное слово, связанное с ними, а иногда нет. Порядок и написание слов в мнемонической фразе напрямую связаны с закрытым ключом, поэтому такой способ не рекомендуется для регулярного доступа к кошельку. Эту фразу лучше использовать только для восстановления.



## __Подключение с помощью мнемонической фразы (и доп. слова):__
#### __Это не рекомендуемый способ подключения. Мы настоятельно рекомендуем [использовать MEW оффлайн](/@@@@@@/offline/offline-mew-looks-weird/) для этого варианта.__

**Шаг 1.** На главной странице MEW выберите "Перейти к моему кошельку".

**Шаг 2.** Выберите вариант "ПО" для подключения, затем выберите "Мнемоническая фраза" и нажмите "Продолжить".

**Шаг 3.** Выберите, сколько слов во фразе, 12 или 24, а затем заполните форму.

**Шаг 4.** Введите дополнительное слово (оставьте это поле пустым, если вы используете фразу из MEWconnect) и нажмите «Продолжить».

**Шаг 5.** Выберите адрес вашего кошелька из сгенерированного списка, готово!



## __Закрытый ключ__
Закрытый ключ — это строка из 64 символов, неразрывно связанная с публичным адресом. Это то, что дается пользователям при создании кошельков и иногда хранится в зашифрованном виде (в случае MEWconnect, MetaMask и аппаратных кошельков). *Закрытый ключ должен использоваться для входа только в исключительных случаях, когда нет другого способа входа.* Этот способ входа не должен быть основным способом доступа к кошельку. Закрытые ключи — это самые ценные сведения, относящиеся к кошельку, потому что они дают непосредственный доступ и не могут быть изменены. Поэтому крайне важно хранить их в тайне, в надежном и защищенном месте.



## __Подключение с помощью закрытого ключа:__
#### __Это не рекомендуемый способ подключения. Мы настоятельно рекомендуем [использовать MEW оффлайн](/@@@@@@/offline/offline-mew-looks-weird/) для этого варианта.__

**Шаг 1.** На главной странице MEW выберите "Доступ к кошельку".

**Шаг 2.** Выберите вариант "Программный доступ" для подключения, затем выберите "Закрытый ключ" и нажмите "Продолжить".

**Шаг 3.** Скопируйте и вставьте ваш закрытый ключ в поле.

**Шаг 4.** Нажмите "Открыть кошелек", готово!
